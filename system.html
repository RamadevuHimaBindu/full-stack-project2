üìñ Concept Explanations

Load Balancing

Distributes incoming network traffic across multiple servers to ensure no single server bears too much load.

Example: Using Nginx or HAProxy to distribute traffic to 3 backend servers.

Caching

Caching stores frequently accessed data in memory to reduce database load and improve response times.

Tools: Redis, Memcached
